
::: vim tmux.conf

Cambiar el prefijo tmux de Ctrl + B a Ctrl + A agregando la siguiente línea en el archivo tmux.conf .


set -g prefix2 C-a                        # GNU-Screen compatible prefix
bind C-a send-prefix -2

set -g prefix C-k
unbind C-b
